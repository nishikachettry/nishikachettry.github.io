Aston Tracking System (ATS)

This group project was one of the most interesting and complicated to achieve since I was not much acquainted with the technology that we used.

What is ATS?

ATS is an RFID security system developed to protect the expensive components in the laboratories from unauthorised access. It also has its very own SAS (Security Alarm System) which activates in case someone tries to gain unauthorised access. The system consists of a component reader, a card scanner for authorised people to gain access and a security system.

Planning

Initially, we were planning to work on a different idea for this group project but after a couple of weeks we realised it was technically too complicated to achieve in the limited timeframe and would be out of budget. Therefore, we had to think of another idea. At this point, I suggested an idea to create a security system for the expensive components in the laboratories. This idea was improved further by adding the security system and other features. I planned the prototyping stages for the system. We used basecamp, gantt chart and a checklist for project management. I regularly updated these for being able to keep track of the progress.

Learnings and Improvements

This project was an excellent learning experience. I familiarised myself with a new technology and learnt about the different aspects of product development. I did not only participate in the technical aspects but also gained an insight into the finance and marketing of a product. Alongside these technical skills, I was also able to develop my soft skills such as communication, presentation and team-work skills. I was also able to learn more about project management which I believe is an essential skill.

Next time, I would work on the finance alongside the technical development as that would reduce the pressure of having to complete it together at the end.
